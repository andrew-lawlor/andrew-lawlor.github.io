What is LibrePub?

LibrePub is a simple, text-focused eBook format that prioritizes ease of creation, reading, and openness.

  • Simple plain-text chapters
  • Open, text-forward format
  • Image support
  • Lightweight

LibrePub Reader

The official LibrePub Reader is a free and open-source application designed to provide a distraction-free reading experience with powerful features:

  • Annotations and highlights
  • Local and web dictionary lookup (for now, only in English)
  • Configurable fonts, sizes, and colors
  • Vocabulary builder with CSV (Anki-compatible) flashcard export
  • Auditable, transparent code. No data collection. No ads
  • Available for Android, Linux, and Windows

Creating LibrePub Ebooks

Getting started with LibrePub is easy! Here's a quick guide to creating your own LibrePub ebooks:

  1. Create a new folder: Ensure there are no spaces in the file name.
  2. Prepare your text content: Each chapter of your ebook should be a separate plain text (.txt) file. Store all chapter files within a folder named "chapters". You can use any plain text editor to create your chapters. LibrePub itself doesn't enforce any specific formatting -- express yourself!
  3. Cover Image: Create an "images" folder under your parent directory. Place your cover image inside.
  4. Optional Images:You may place other images inside the image folder. Apps that read LibrePub files can allow the user to browse them. Images can also be inserted into the text body in the following way: LIBREPUB_IMAGE(image_file_name.png)
  5. Create a manifest.json file: This file provides essential metadata about your ebook. Just create a simple text file named "manifest.json" in your root directory and add the following entries, replacing the bracketed placeholders with your actual information: 
    
         {  
            "title": "[Your Ebook Title]",
            "author": "[Your Name]",
            "language": "[en]",
            "genre": "[Your Genre]"
            "cover_image": "[cover.jpg]"
          }
    The "title", "author", "cover_image" and "language" fields are required. "genre" is optional.
  6. Zip it Up: Place your "chapters" folder, the "images" folder (if you have images), and the "manifest.json" file into the root of a single zip archive. Rename the zip archive using the `.lpub` extension (e.g., "your_ebook.lpub").

Congratulations! You've created your first LibrePub ebook.
