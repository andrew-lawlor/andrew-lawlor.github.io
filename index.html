What is LibrePub?

LibrePub is a simple, text-focused ebook format that prioritizes ease of creation, reading, and openness.

  • Simple plain text chapters
  • Image support
  • Essential metadata (title, author, and language)
  • Optional advanced metadata (genre, cover image)

LibrePub Reader

The official LibrePub Reader is a free and open-source application designed to provide a distraction-free reading experience with powerful features:

  • Annotations and highlights
  • Local and web dictionary lookup (for now, only in English)
  • Configurable fonts, sizes, and colors
  • Vocabulary builder with CSV (Anki-compatible) flashcard export
  • Auditable, transparent code. No data collection. No ads
  • Available for Android, Linux, and Windows

Creating LibrePub Ebooks

Getting started with LibrePub is easy! Here's a quick guide to creating your own LibrePub ebooks:

  1. Prepare your text content: Each chapter of your ebook should be a separate plain text (.txt) file. Store all chapter files within a folder named "chapters". You can use any plain text editor to create your chapters. LibrePub itself doesn't enforce any specific formatting within the chapters.
  2. Optional Images: If you want to include images in your ebook, place them in a folder named "images". These images will be browseable by within LibrePub Reader or other apps that support the LibrePub format.
  3. Create a manifest.json file: This file provides essential metadata about your ebook. Just create a simple text file named "manifest.json" and add the following entries, replacing the bracketed placeholders with your actual information: 
    
         {  
            "title": "Your Ebook Title",
            "author": "Your Name",
            "language": "en",  /* Change language code if needed */
            "genre": "Your Genre"  /* Optional */
            "cover_image": "cover.jpg" /* Optional */
          }
    The "title", "author", and "language" fields are required.
  4. Zip it Up: Place your "chapters" folder, the "images" folder (if you have images), and the "manifest.json" file into the root of a single zip archive. Rename the zip archive using the `.lpub` extension (e.g., "your_ebook.lpub").

Congratulations! You've created your first LibrePub ebook.
